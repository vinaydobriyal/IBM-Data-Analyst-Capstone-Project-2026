# Stack Overflow Developer Survey Analysis

Analysis of the Stack Overflow Developer Survey dataset, exploring trends across databases, programming languages, web frameworks, cloud platforms, 
and developer demographics. The project includes a full written report and a presentation deck summarizing key findings.

## 📌 Project Overview

This project analyzes developer survey data to uncover trends in technology adoption and developer preferences. 
The goal is to identify patterns across tools, platforms, and demographics that reflect the current state of the software development industry.

## 🎯 Objectives

- Identify the most popular databases, programming languages, and web frameworks
- Analyze cloud platform adoption trends
- Explore developer demographics (experience, region, role, etc.)
- Present findings in a structured, business-friendly format (findings + implications)

## 🛠️ Tools & Technologies

- **Python** (Pandas) — data cleaning, transformation, and analysis | 
             (matplotlib and seaborn) - Data visualization charts and graphs
- **SQL** — querying and aggregating survey data
- **Cognos Analytics** — interactive dashboards and visualizations
- **Excel** — supplementary data exploration
- **Powerpoint** — Slide based Presentation Report

## 🔍 Key Areas of Analysis

1. **Databases** — most used and most desired databases among developers
2. **Programming Languages** — popularity trends and year-over-year shifts
3. **Web Frameworks** — adoption rates across frontend/backend frameworks
4. **Cloud Platforms** — usage share among major providers
5. **Developer Demographics** — experience levels, roles, and regional trends

## 📊 Deliverables

## 📈 Key Findings

- JavaScript leads current usage (16k+).
- PostgreSQL dominates (10k+).
- AWS is the clear leader (10,871 users).
- React & Node.js dominate both current usage and future desire.
- 25–34 age group is the largest cohort (7,788). Bachelor's degree is the most common education level.

## Implications

- Prioritize mastering TypeScript & Python for future-proofing.
- Adopt PostgreSQL as the default database choice.
- Embrace Next.js & React for frontend development.
- Go cloud-native with AWS + Azure, which together cover 60%+ of market demand.
- Keep an eye on Rust, Supabase & FastAPI as rising technologies.
- Target the 25–34 age segment when building products or content for developers.
